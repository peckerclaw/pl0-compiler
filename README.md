# PL/0 编译器扩充实现

## 项目概述

**源语言：** PL/0（Pascal 子集）  
**目标语言：** PL/0 字节码（.cod）  
**实现语言：** C 语言（跨平台 ANSI C）  
**原版来源：** 清华大学出版社《编译原理（第3版）》附录A

## 扩充内容

### 课内实验（考试前交报告）

| 序号 | 扩充内容 | 状态 |
|------|---------|------|
| 1 | 增加保留字：`FOR`, `TO`, `DOWNTO`, `RETURN` | ✅ |
| 2 | 增加运算符：`+=`, `-=`（赋值扩展） | ✅ |
| 3 | 增加运算符：`++`, `--`（表达式级别） | ✅ |
| 4 | 替换不等号：`#` → `<>` | ✅ |

## 项目结构

```
pl0-compiler/
├── src/
│   └── pl0.c          # 扩充后的 PL/0 编译器源码
├── tests/
│   ├── 01_basic.pl0       # 基础测试（原始 PL/0 功能）
│   ├── 02_relop.pl0       # 不等号 <> 测试
│   ├── 03_if_else.pl0    # IF-ELSE 语句测试
│   ├── 04_for_to.pl0      # FOR TO 语句测试
│   ├── 05_for_downto.pl0  # FOR DOWNTO 语句测试
│   ├── 06_compound.pl0    # += -= 测试
│   ├── 07_inc_dec.pl0     # ++ -- 测试
│   ├── 08_real.pl0        # 实数类型测试
│   ├── 09_char.pl0        # 字符类型测试
│   ├── 10_function.pl0   # 函数 + RETURN 测试
│   ├── 11_array.pl0       # 一维数组测试
│   └── 12_integrated.pl0  # 综合测试
├── docs/
│   ├── grammar.md         # 扩充文法说明
│   ├── syntax_charts.md   # 语法图说明
│   └── semantic_rules.md  # 语义规则说明
├── Makefile
└── README.md
```

## 编译运行

```bash
make          # 编译
make test     # 运行所有测试
./bin/pl0 tests/01_basic.pl0   # 编译运行单个文件
```

## 保留字总表

原始：`BEGIN CALL CONST DO END IF ODD PROCEDURE PROGRAM READ THEN VAR WHILE WRITE`

扩充：`FOR TO DOWNTO RETURN PE(+=.赋值) ME(-=.赋值) PP(++.) MM(--.) REAL CHAR ARRAY FUNCSYM`

## 运算符

| 运算符 | 含义 |
|-------|------|
| `+` `-` `*` `/` | 四则运算 |
| `=` `<>` `<` `<=` `>` `>=` | 关系运算 |
| `:=` | 赋值 |
| `+=` `-=` | 复合赋值 |
| `++` `--` | 自增自减（表达式级） |
